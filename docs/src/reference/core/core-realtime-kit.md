# @privateaim/core-realtime-kit

WebSocket event types and utilities for real-time client-server communication in PrivateAIM.

## Installation

```bash
npm install @privateaim/core-realtime-kit
```

## Usage

```typescript
import {
    DomainEventChannelName,
    buildDomainChannelName,
} from '@privateaim/core-realtime-kit';
```

## API

### Exports

| Module | Description |
|--------|-------------|
| `constants` | Event channel name constants |
| `types` | WebSocket event type definitions |
| `utils` | Channel name building utilities |

## Dependencies

- `@privateaim/kit` — Core utilities
- `@privateaim/core-kit` — Domain types
