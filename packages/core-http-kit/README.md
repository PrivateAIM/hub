# @privateaim/core-http-kit 🔗

[![npm version][npm-src]][npm-href]

Typed HTTP client for the PrivateAIM core API, built on [Hapic](https://github.com/tada5hi/hapic).

## Installation

```bash
npm install @privateaim/core-http-kit
```
